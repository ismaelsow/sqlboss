How long do I have access to the program and its material?

You get lifetime access to all of the program's resources and material: video recordings, lessons, exercises with solutions, extra-resources etc.

What flavor of SQL will I learn?

We'll be using the Google Standard SQL dialect on the Google BigQuery database. It shouldn't matter much as the syntax variations that exist between database engines (Snowflake, Oracle, PostgreSQL, MySQL, Microsoft SQL Server, etc.) are small and easy to pick up.

Does the program cover database design, modification or administration topics?

No. The program focuses on analysis and reporting, i.e answering business questions with SELECT-based queries. We cover user-defined functions, table functions, and views as they can be used to make better SELECT-based queries. We don't include anything that create, update or delete database records.

What equipment and software do I need?

Many teaching programs, courses and books ask you to install some desktop database software and toy datasets. We don't. There are too many horror stories of students unable to install the software and getting stuck. For the database, we use Google BigQuery. The live classes are taught over Google Meet. Both BigQuery and Meet have free plans and are browser-based. Bottom line: you only need a computer with an internet connection, and a Gmail account.

What will be the outcome of the program?

After completing the program, you'll be comfortable reading and writing advanced SQL queries for any of the following roles: data analyst, BI analyst, marketing analyst, product analyst, business analyst, web analyst, etc.
