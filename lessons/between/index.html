We can use the BETWEEN operator to create interval-based conditions.

Example 1

Type the query below in the BigQuery console and run it. Don't copy paste. 😉

  
SELECT  *
  FROM  bigquery-public-data.thelook_ecommerce.orders
  WHERE order_id BETWEEN 1 AND 5;

N°1 — 10 orders created in September 2022  🌶️

Tables

bigquery-public-data.thelook_ecommerce.orders

Task

Write a query that displays 10 orders created in september 2022. You can use the format you see in the created_at column for your date condition.

Example results

Solution

🙈 Hide👀 Show

        
SELECT  order_id, created_at, status, num_of_item, gender
  FROM  bigquery-public-data.thelook_ecommerce.orders
  WHERE created_at BETWEEN '2022-09-01 00:00:00 UTC' AND '2022-09-30 23:59:59 UTC'
  LIMIT 10;
