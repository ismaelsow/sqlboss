The OFFSET clause helps us paginate the result set returned by our SELECT. It has to be used together with the LIMIT clause. So the following code:

  
SELECT      id, name, retail_price
  FROM      bigquery-public-data.thelook_ecommerce.products
  ORDER BY  retail_price DESC
  LIMIT     10
  OFFSET    90;
