We can also use the OR keyword to combine multiple conditions in our WHERE clause. We use the OR keyword to specify conditions where AT LEAST ONE needs to be true

Example 1

Type the query below in the BigQuery console and run it. Don't copy paste. 😉

  
SELECT  id, first_name, last_name, email, state, city, country
  FROM  bigquery-public-data.thelook_ecommerce.users
  WHERE country = 'Austria' OR city = 'Vienna'
  LIMIT 10;

N°1 — Products containing ‘dna’ in their name  🌶️

Tables

bigquery-public-data.thelook_ecommerce.products

Task

Write a query that displays all products that contain ‘dna’ in their name, regardless of the case.

Example results

Solution

🙈 Hide👀 Show

        
SELECT  id,
        name
  FROM  bigquery-public-data.thelook_ecommerce.products
  WHERE name LIKE '%dna%'
        OR name LIKE '%Dna%'
        OR name LIKE '%DNa%'
        OR name LIKE '%DNA%'
        OR name LIKE '%dNa%'
        OR name LIKE '%dNA%'
        OR name LIKE '%dnA%'
        OR name LIKE '%DnA%';
