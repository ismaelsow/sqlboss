About BigQuery and Google Standard SQL

  • BigQuery is a database built by Google, accessible in the cloud.
  • It supports SQL queries, using Google Standard SQL dialect (a standard-compliant flavor of SQL).
  • It's fast, scalable and cost-effective (offering a generous free plan).
  • It's often used to build data warehouses in the cloud.
