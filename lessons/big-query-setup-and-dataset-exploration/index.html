About BigQuery and Google Standard SQL

  • BigQuery is Google’s cloud data warehouse solution
  • BigQuery is fast, scalable and cost-effective
  • BigQuery supports the Google Standard SQL dialect which is an ANSI compliant flavor of SQL

Establishing the BigQuery environment

  • Create a Gmail account if don’t have one
  • Log into your Gmail account
  • Go to https://console.cloud.google.com/bigquery
  • If you don’t already have a Google Cloud Project created, you’ll be prompted to create one. Click Create Project and follow the prompts.
  • There are two versions of the Google BigQuery environment. The first is the BigQuery sandbox, a feature-limited, no-cost option for playing around with the tool.
  • The other one is the upgraded environment, for which you’ll need to provide your credit card.
  • As instructors, we’ll be using the upgraded environment, but start with the sandbox instead.

Monitoring BigQuery costs

If you grow out of the BigQuery sandbox, you’ll need to create a billing account and keep an eye on your BigQuery usage in order to avoid surprise bills. To do so open this link: https://console.cloud.google.com/billing

When your billing account is created, click on your billing account from this same link to see your cost and cost projections. Bookmark that page and check it often.
