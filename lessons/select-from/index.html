Task

Write a query that will display:

  • the id, name, cost, retail_price columns
  • Plus a new column named margin that will show the difference between the retail_price and the cost
  • Plus a new column called retail_price_increased that shows retail prices increased by 10%
  • And another column called retail_priced_increased_margin that will show the new margin with this 10% increase.

Example results

Solution

ðŸ™ˆ HideðŸ‘€ Show

        
SELECT  id,
        name,
        cost,
        retail_price,
        retail_price - cost AS gross_margin,
        retail_price * 1.1 AS retail_price_increased,
        (retail_price * 1.1 - cost) AS retail_price_increased_margin
  
  FROM  bigquery-public-data.thelook_ecommerce.products;
