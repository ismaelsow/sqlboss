How to use the course

Here are five simple rules you should live by:

  1. Read everything
  2. Run all the examples
  3. Do all the exercises
  4. Don't look at the solution too soon
  5. Type the code snippets by hand (don't copy-paste)

Take these rules seriously. They probably come from an ancient Greek stoic, or something.
