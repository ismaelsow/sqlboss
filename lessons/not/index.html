The NOT keyword is used to specify a “negative condition”.

Example 1

  
SELECT  *
  FROM  bigquery-public-data.thelook_ecommerce.distribution_centers
  WHERE name NOT IN ('Memphis TN', 'Savannah GA');

Example 2

  
SELECT  email
  FROM  bigquery-public-data.thelook_ecommerce.users
  WHERE email NOT LIKE '%@example.org'
  LIMIT 10;

Example 3

Since we can’t use normal operators or NULL, when we want to test that something has a value, we use the ‘IS NOT NULL’ construct.

  
SELECT  *
  FROM  bigquery-public-data.thelook_ecommerce.orders
  WHERE returned_at IS NOT NULL
  LIMIT 10;

N°1 — 10 products that don’t have “Men’s” in their name  🌶️

Tables

bigquery-public-data.thelook_ecommerce.products

Task

Write a query that will show 10 products that don’t have Men’s in their name.

Example results

Solution

🙈 Hide👀 Show

        
SELECT  name
  FROM  bigquery-public-data.thelook_ecommerce.products
  WHERE name NOT LIKE '%Men\'s%'
  LIMIT 10;
