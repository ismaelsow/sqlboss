The UNION ALL operator is used to add the result of queries on top of each other. The result sets of each query must have the same number of columns, with the same types, and in the same order.

Tables

Task

Write a query that will display the number of unique states that have events OR users.

Example results

Solution

üôà HideüëÄ Show

        
WITH all_states AS
  (   SELECT  state
        FROM  bigquery-public-data.thelook_ecommerce.events
      
      UNION ALL
  
      SELECT  state
        FROM  bigquery-public-data.thelook_ecommerce.users
  )

SELECT  DISTINCT state
  FROM  all_states;

  • Create a new BigQuery dataset and call it dbpedia.

WITH l3_categories AS
  ( SELECT  DISTINCT l3 AS category,
            l2 AS parent
      FROM  dbpedia.pages
  ),
  
  l2_categories AS
    (  SELECT DISTINCT l2 AS category,
              l1 AS parent
        FROM  dbpedia.pages
    ),
  
  l1_categories AS
    (  SELECT DISTINCT  l1 AS category,
                        '' AS parent
                  FROM  dbpedia.pages
    ),
  
  all_categories AS
    ( SELECT  *
        FROM  l1_categories
      
      UNION ALL
      
      SELECT  *
        FROM  l2_categories
      
      UNION ALL
      
      SELECT  *
        FROM  l3_categories
    )

SELECT  *
  FROM  all_categories;
